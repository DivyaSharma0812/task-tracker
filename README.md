# Task Tracker ✅
A simple Task Tracker web app that lets you:
- Add tasks
- View all tasks
- Mark tasks as completed (and undo)

Tasks persist using **Local Storage**, so they remain after refresh.

## Run Locally
### Option 1: Open directly
Just open `index.html` in your browser.

### Option 2: Use a local server (recommended)
If you have VS Code:
- Install **Live Server**
- Right-click `index.html` → **Open with Live Server**
